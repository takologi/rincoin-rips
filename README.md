# RIPs (Rincoin Improvement Proposals)

People wishing to submit a RIP should first describe their idea to the community (e.g., Discord or Developer channels) to gather feedback on viability and community interest before working on a formal description.

Please open a pull request to this repository only when substantial progress on the draft has been made, preferably when the draft is nearing completion.

Authors do *not* assign a number to their own proposal. After a proposal meets the editorial criteria, a RIP Editor will assign a number to it and publish the proposal by merging the pull request.

The RIPs repository serves as a publication medium and archive. Having a RIP published here indicates that the proposal is in scope and has met formal criteria, but does not indicate that it is a good idea, has community consensus, or that it is about to be adopted.
